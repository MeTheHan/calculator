NovaCalc Pro - Yasal Bilgiler

Genel Bakış

NovaCalc Pro, modern ve kullanıcı dostu arayüzüyle öne çıkan, web tabanlı gelişmiş bir hesap makinesi uygulamasıdır. Hem temel matematik işlemleri hem de bilimsel hesaplamalar için tasarlanmıştır.

Uygulama, kullanıcı deneyimini ön planda tutarak geliştirilmiş olup, responsive tasarımı sayesinde tüm cihazlarda sorunsuz çalışmaktadır.

Özellikler

Temel Özellikler

  • Temel İşlemler: Toplama, çıkarma, çarpma, bölme
  • Bilimsel Fonksiyonlar: Trigonometri, logaritma, üstel fonksiyonlar
  • Hafıza İşlevleri: MC, MR, M+, M-, MS
  • İşlem Geçmişi: Son 10 işlemi kaydetme ve görüntüleme
  • Çok Dilli Destek: Türkçe ve İngilizce
  • Tema Seçenekleri: Açık/Koyu tema

Teknik Özellikler

  • Responsive Tasarım: Mobil uyumlu
  • Klavye Desteği: Klavye kısayolları
  • Yerel Depolama: Ayarlar ve geçmiş tarayıcıda saklanır
  • Ses Efektleri: Buton tıklama sesleri
  • Animasyonlar: Yumuşak geçişler ve efektler

Teknik Detaylar

Teknoloji Stack

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Stil: CSS Variables, Flexbox, Grid
  • İkonlar: Font Awesome 6.4.0
  • Depolama: LocalStorage API
  • Ses: Web Audio API

Tarayıcı Uyumluluğu

  • ✅ Chrome 90+
  • ✅ Firefox 88+
  • ✅ Safari 14+
  • ✅ Edge 90+

Kullanım

Temel Kullanım

  1. Sayıları girmek için sayı butonlarını kullanın
  2. İşlemler için operatör butonlarını tıklayın
  3. Sonucu görmek için "=" butonuna basın

Klavye Kısayolları

  • Sayılar: 0-9
  • Operatörler: +, -, *, /
  • Enter: Hesapla
  • Escape: Temizle
  • Backspace: Sil

Ayarlar

  • ⚙️ Ayarlar butonundan dil ve tema değiştirilebilir
  • 🔇 Ses efektleri açılıp kapatılabilir
  • 🎭 Animasyonlar etkinleştirilebilir
