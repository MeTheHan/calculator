## 🚀 Overview

NovaCalc Pro is a sophisticated web-based calculator that combines elegant design with powerful functionality. Built with modern web technologies, it offers both basic and scientific calculations in a user-friendly interface.

![Calculator Preview](https://via.placeholder.com/800x400/2a2a3c/ffffff?text=NovaCalc+Pro+Modern+Calculator)

## ✨ Features

### 🎯 Core Functionality
- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**: Trigonometry, logarithms, exponents, roots
- **Memory Operations**: MC, MR, M+, M-, MS
- **Calculation History**: Stores last 10 calculations
- **Dual Language Support**: English and Turkish
- **Theme Options**: Light/Dark mode

### 🛠 Advanced Features
- **Responsive Design**: Mobile-friendly interface
- **Keyboard Support**: Full keyboard shortcuts
- **Local Storage**: Saves settings and history
- **Sound Effects**: Tactile feedback
- **Smooth Animations**: Beautiful transitions
- **Decimal Precision Control**: 0-8 decimal places

## 🛠 Installation

### Prerequisites
- Modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- JavaScript enabled
- LocalStorage support

### Quick Setup
1. Clone or download this repository
```bash
git clone https://github.com/MeTheHan/novacalc-pro.git
```
2. Open `index.html` in your web browser

## 📱 Usage

### Basic Operations
1. Click number buttons or use keyboard (0-9)
2. Select operation (+, -, ×, ÷)
3. Press equals (=) or Enter for result

### Keyboard Shortcuts
| Key | Function |
|-----|----------|
| `0-9` | Number input |
| `+ - * /` | Basic operations |
| `Enter` or `=` | Calculate |
| `Escape` or `Delete` | Clear |
| `Backspace` | Delete last digit |
| `.` or `,` | Decimal point |

### Settings
- ⚙️ **Settings Button**: Access language and theme options
- 🔇 **Sound Toggle**: Enable/disable sound effects
- 🎭 **Animations**: Toggle UI animations
- 🌐 **Language**: Switch between English and Turkish
